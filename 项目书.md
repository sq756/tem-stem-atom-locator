# 原子级 TEM/STEM 图像原子柱位点识别工具项目书

## 1. 项目名称

原子级 TEM/STEM 图像中原子柱位点的原理驱动识别与晶格关系分析工具

## 2. 项目背景

高分辨透射电子显微镜图像，尤其是 HRTEM、STEM-HAADF、ABF 等模式，可以直接反映材料中原子柱或投影原子位点的空间分布。对这些位点进行可靠提取，是后续晶格畸变分析、缺陷识别、相结构判定、界面分析和元素占位推断的基础。

当前常见的图像识别方法倾向于直接使用神经网络进行端到端检测或分类，但这类方法通常需要大量标注数据，且解释性不足。对于本项目的第一阶段，更合理的路线是从图像形成和晶体几何关系出发，将原子柱识别建模为局部峰值检测、亚像素定位和点阵几何分析问题，先建立一个可解释、可调参、可验证的原子位点检测器。

本项目将以传统图像处理和晶格几何分析为核心，构建一个能够处理不同比例尺、不同尺寸、不同清晰度 TEM/STEM 图像的原子柱位点识别工具。该工具在第一阶段不直接判断元素种类和物相，而是稳定输出原子柱候选点、亚像素坐标、局部强度、峰宽、邻近点关系和置信度，为后续相识别和缺陷分析提供结构化数据基础。

## 3. 项目目标

本项目第一阶段目标是实现一个面向 TEM/STEM 原子级图像的原子柱位点识别原型工具，具体包括：

1. 支持读取 `.tif/.tiff` 格式的 TEM/STEM 图像。
2. 对不同亮度、背景、噪声和清晰度的图像进行标准化预处理。
3. 基于局部极值、多尺度 LoG/DoG blob 检测和局部对比度筛选，提取原子柱候选点。
4. 使用二维高斯拟合或局部质心法实现亚像素级位点精修。
5. 建立候选点之间的最近邻关系，统计原子间距、角度分布和局部配位关系。
6. 输出结构化结果，包括原子柱坐标、强度、峰宽、置信度、邻居列表和可视化标注图。
7. 为后续相识别、缺陷识别和元素占位分析预留接口。

## 5. 技术路线

项目采用“图像标准化 -> 原子柱候选检测 -> 亚像素定位 -> 几何关系分析 -> 结构化输出”的技术路线。

```text
TEM/STEM 原始图像
  -> 灰度读取与强度归一化
  -> 背景扣除与去噪
  -> 多尺度局部峰检测
  -> 候选点筛选
  -> 亚像素坐标精修
  -> 最近邻和晶格向量分析
  -> 原子位点表、邻接关系和可视化结果
```

### 5.1 图像标准化

不同 TEM/STEM 图像可能存在比例尺不同、亮度不同、背景不均匀、局部漂移、噪声水平不同等问题。预处理模块需要将图像转换为适合统一检测的形式。

主要方法包括：

- 灰度图读取，保留原始位深信息。
- 强度归一化，使用 percentile clipping 抑制极端亮暗值。
- 背景校正，使用大尺度 Gaussian blur 或 rolling-ball 类方法估计慢变化背景。
- 去噪处理，优先使用 Gaussian blur、median filter、bilateral filter 等可解释方法。
- 支持亮斑模式和暗斑模式；暗斑图像可先反相再进入统一检测流程。

### 5.2 原子柱候选点检测

原子柱在图像上通常表现为局部亮峰或暗峰。第一阶段将其建模为局部 blob 检测问题。

候选检测方法包括：

- 局部极大值检测。
- Difference of Gaussian 检测。
- Laplacian of Gaussian 多尺度 blob 检测。
- 最小距离约束，避免一个原子柱被重复检测。
- 峰值强度、局部对比度、峰宽和响应值筛选。

多尺度检测是核心，因为不同放大倍率、不同像素尺寸和不同清晰度下，原子柱在图像中的 apparent radius 不一致。算法应允许在多个 `sigma` 上搜索响应峰，并记录最佳尺度。

### 5.3 亚像素位点精修

候选点的整数像素位置不足以支持可靠的晶格距离和角度分析。每个候选点周围取局部窗口，进行位置精修。

可采用两类方法：

- 局部强度加权质心法，计算速度快，适合作为默认快速方法。
- 二维高斯拟合法，输出中心坐标、峰强度、峰宽、椭圆率和拟合残差，适合作为高精度方法。

输出参数包括：

```text
x_px, y_px
amplitude
local_background
sigma_x, sigma_y
ellipticity
fit_error
confidence
```

### 5.4 点阵关系分析

完成位点检测后，需要判断这些点是否形成合理的晶格结构，并为后续相识别提供几何基础。

主要分析包括：

- 使用 KD-tree 查找每个点的最近邻。
- 统计最近邻距离分布，估计典型原子柱间距。
- 统计点对角度分布，估计主晶格方向。
- 构建 Delaunay triangulation 或邻接图。
- 标记异常位点，例如距离异常、配位数异常、强度异常或边缘不可靠点。

第一阶段不强行判断物相，而是输出晶格几何特征，为后续相识别模块使用。

## 6. 系统架构

建议先采用 Python 原型架构，待算法稳定后再考虑迁移到 Swift/iOS/macOS 工具中。

```text

  data_io.py          # TIFF 读取、元数据、图像格式转换

```


## 8. 评价指标

第一阶段评价重点不是分类准确率，而是位点检测质量。

可使用以下指标：

- 检测点是否覆盖主要原子柱。
- 是否存在明显漏检。
- 是否存在噪声误检。
- 重复检测比例。
- 亚像素位置是否落在峰中心。
- 最近邻距离分布是否形成稳定峰。
- 对不同清晰度和不同尺度图像的参数鲁棒性。

如果后续有人工标注点，可以进一步计算：

- precision
- recall
- F1 score
- mean localization error
- nearest-neighbor structure consistency

## 10. 风险与应对

### 图像质量差异大

风险：不同图像的噪声、模糊、背景和对比度差异可能导致统一参数失效。

应对：采用多尺度检测和参数配置文件，并在预处理阶段加入背景扣除、局部对比度归一化和亮斑/暗斑模式选择。

### 原子柱重叠或峰形畸变

风险：缺陷、界面、厚度变化或像差会导致峰形变宽、偏移或重叠。

应对：在高斯拟合中记录椭圆率和残差，将异常峰标记为低置信度，不在第一阶段强行分类。

### 缺少人工标注数据

风险：难以量化检测准确率。

应对：先用 overlay 图进行人工快速检查，并逐步建立少量人工标注点作为验证集。

### 过早引入复杂模型

风险：神经网络模型可能掩盖图像处理和几何判断中的基础问题。

应对：第一阶段坚持使用可解释算法，待位点检测和几何特征稳定后，再考虑是否引入机器学习用于分类或参数自适应。

## 11. 后续扩展方向

- 相结构识别。

## 12. 第一版实施原则

第一版工具应遵循以下原则：

1. 先检测点，不急于识别元素或相。
2. 每一步输出中间图，便于判断错误来源。
3. 所有参数可配置，不写死在代码里。
4. 输出结构化数据，为后续算法复用。
5. 优先使用可解释方法，保留物理和几何意义。
6. 神经网络只作为后续增强手段，不作为第一阶段核心依赖。
